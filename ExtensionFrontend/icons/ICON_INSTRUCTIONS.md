# Extension Icons

You need to create three icon files for the Chrome extension:

1. **icon16.png** - 16x16 pixels (toolbar icon)
2. **icon48.png** - 48x48 pixels (extension management page)
3. **icon128.png** - 128x128 pixels (Chrome Web Store)

## Quick Creation Options

### Option 1: Use Online Icon Generator
1. Go to https://www.favicon-generator.org/ or similar
2. Upload a logo/image
3. Download the required sizes
4. Save as `icon16.png`, `icon48.png`, `icon128.png` in this folder

### Option 2: Create Simple Icons
1. Create a simple design with the letter "P" for PriceCompare
2. Use a blue gradient background matching your brand
3. Export at the three required sizes

### Option 3: Use Placeholder
You can use a simple colored square for now:
- Blue square with "P" text
- Same design as used in the web app logo

## Recommended Design
- Background: Blue gradient (matching web app)
- Text: White letter "P" or a price tag icon
- Border radius: 8px (rounded corners)
